
Ideal Fishing - Netlify CMS-ready site (Romanian)

To enable Netlify CMS editing:
1. Push this repository to GitHub (main branch).
2. On Netlify create a new site from that GitHub repo.
3. In Netlify site settings: enable Identity (open registration optionally) and enable Git Gateway.
4. Deploy site. Access CMS at https://<yoursite>.netlify.app/admin

Replace {your_form_id} in the contact form if using Formspree, or configure Netlify Forms.
